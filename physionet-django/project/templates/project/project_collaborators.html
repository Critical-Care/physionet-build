Project Collaborators

Roles

Each project can have many collaborators. Collaborators can view and edit project metadata and files.

Each project has one owner, who is also a collaborator. The owner may:

  • Submit the project for publication
  • Delete the project
  • Invite and remove collaborators
  • Assign ownership of the project to another collaborator
