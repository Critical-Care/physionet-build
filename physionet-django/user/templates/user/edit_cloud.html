User Cloud information

Some of our databases are on Google Cloud Platform or Amazon Web Services.

To gain access to them:

  1. Enter your google email address and/or your AWS numerical ID.
  2. Go to a published database and above the files click "Request access".

Note: If the database does not have a "Request access", then that database is not available in the cloud at the moment.
