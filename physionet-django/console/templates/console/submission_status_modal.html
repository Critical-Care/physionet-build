Submission Statuses
  1. Presubmission - submitting author submits project. Awaiting co-authors.
  2. Submitted - all co-authors agree to submission. Awaiting editor assignment.
  3. Submitted - editor is assigned. Awaiting editor decision. Continues to 4, 5, or 6.
  4. Resubmission - editor decision 1. Awaiting submitting author to make revisions.
  5. Rejected - editor decision 2. The submission ends.
  6. Accepted - editor decision 3. The project is published and submission ends.
