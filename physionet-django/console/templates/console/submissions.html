Submission Statuses

  1. Presubmission - submitting author submits project. Awaiting co-authors.
  2. Submitted - all co-authors agree to submission. Awaiting editor assignment.
  3. Submitted - editor is assigned. Awaiting editor decision. Continues to 4, 5, or 6.
  4. Resubmission - editor decision 1. Awaiting submitting author to make revisions. Loops back to 3.
  5. Rejected - editor decision 2. The submission ends.
  6. Accepted - editor approved publication. Awaiting submitting author to confirm.
  7. Accepted - submitting author approves publication. Project is published and submission ends.
